# Full Stack Application - Frontend

This is the frontend part of the full-stack application built using React.js. Below are the instructions to set up and run the frontend.

## Getting Started

1. **Clone the repository:**
   ```
   git clone <repository-url>
   cd full-stack-app/frontend
   ```

2. **Install dependencies:**
   ```
   npm install
   ```

3. **Run the application:**
   ```
   npm start
   ```

   This will start the development server and open the application in your default web browser.

## Folder Structure

- `src/`: Contains all the source code for the frontend application.
  - `App.js`: The main component that sets up routing and layout.
  - `components/`: Contains reusable components.
  - `pages/`: Contains the main pages of the application.
  - `styles/`: Contains CSS styles for the application.

## Usage

You can modify the components and pages as per your requirements. The application is set up to support routing and state management.
